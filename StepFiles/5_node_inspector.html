Node Inspector

At the start of every round, the value of all the nodes are hidden. This makes it difficult to determine the best path for the plane to take. Fortunately, you will have access to a node inspector which can reveal the value of a node. To use the node inspector, simply click on a node. The image below illustrates how this works.

Note: you can only use the node inspector when you're on the first node as pictured in the image below. Once you move the plane, you CANNOT inspect anymore nodes.
